# Overview

The Bruno VS Code extension enables you to test, debug, and send API requests directly from your editor. Designed with a developer-first approach, it simplifies your workflow by allowing you to:

- Manage environments with ease

- Write pre-request and test scripts

- Automate repetitive API operations

- Seamlessly integrate into your development process

You can install the extension from the official [Visual Studio Code Marketplace](https://marketplace.visualstudio.com/items?itemName=bruno-api-client.bruno)
and share your feedback to help us improve.
